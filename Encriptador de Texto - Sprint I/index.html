¿Que es Encriptar?

  • Convertir el contenido de un mensaje en otro código para que pueda ser leído o interpretado en otro momento.
  • La encriptación convierte los datos en texto codificado utilizando una clave secreta: dicho texto ilegible solo se puede decodificar con la clave secreta.
  • Contiene una forma del texto plano original que es ilegible para un ser humano o una computadora sin el cifrado adecuado para descifrarlo. El descifrado es el proceso inverso que convierte un texto cifrado en texto legible.
